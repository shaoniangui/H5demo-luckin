# 项目介绍
为了练习前端技术，模仿的"瑞幸咖啡"APP页面
# 依赖
- 基于 vue.js 2.0 + vue-cli 3.0 + vue-router
- 使用 vant-ui 组件库
- 使用 rem 作为移动端适配方案
- 使用 postcss-pxtorem 插件自动转换
- 使用 mescroll.js 开发上拉加载/下拉刷新
